## Getting Started

1. Clone this project to your local machine

    - HTTPS

    ```console
    git clone https://github.com/Alz3bi/Movies-Library.git
    ```

    - SSH

    ```console
    git clone https://github.com/Alz3bi/Movies-Library.git
    ```

2. install NPM and Node.js, node Express

3. Run the server

    ```console
    npm start
    ```
